# Tests Documentation

## Structure

- [Dynamax T4](./dmax_t4/): Collection of test for T4 Dynamax bosses.
- [Dynamax T5](./dmax_t5/): Collection of test for T5 Dynamax Legendary bosses.
- [Dynamax T6](./dmax_t6/): Collection of test for T6 Dynamax boss Eternatus Eternamax.
- [Gigantamax T6](./gmax_t6/): Collection of test for T6 Gigantamax bosses.
- [Helpers](./help/): Collection of test for all possible Helpers bonus values.
- [Other](./other/): Collection of test for aditional features, such as low tier Dynamax bosses, timers, enrage damage and Shroom bonus.
